# ShopSphere - Production Ready DevOps Project

## Project Overview

ShopSphere is a production-style e-commerce application built to demonstrate real-world DevOps practices.

This project is part of a 7-Day DevOps Challenge where the complete application will be built, containerized, deployed on AWS, and automated using Jenkins CI/CD.

## Technology Stack

- React
- Node.js
- PostgreSQL
- Docker
- Docker Compose
- Jenkins
- Nginx
- AWS EC2
- Git & GitHub
